Listas Descriptivas: Definiciones de Clase

HTML
Lenguaje de marcado para la creación de páginas web.
CSS
Lenguaje utilizado para describir la presentación de un documento HTML.
JavaScript
Lenguaje de programación que permite implementar funcionalidades complejas en páginas web.
Etiqueta
Elemento básico de HTML que define la estructura y el contenido de una página web.
Validación de Web
Proceso de comprobar que el código HTML cumple con las reglas del estándar.

Utilizando blockquote

Esta etiqueta permite delimitar un bloque de texto que por alguna razón (nota, apartado, cita...) se quiere destacar.
